# 3D Architectural Animation (Blender)

## Overview

This project is a **3D architectural modeling and animation** created using Blender. It showcases a **classical-style structure with columns** placed on a terrain base, along with an animated walkthrough to present the model from multiple perspectives.

The goal of this project was to explore **3D modeling, scene composition, and animation techniques** while building a visually appealing architectural structure.

---

## Features

*  **Architectural Modeling**
  Designed a symmetric classical structure with multiple columns and elevated platform.

*  **Environment Setup**
  Created a terrain base with layered materials to simulate grass and ground.

* **Animation**
  Implemented a camera-based animated walkthrough using keyframes to showcase the model.

* **Lighting & Rendering**
  Applied lighting techniques and materials to enhance depth, realism, and visual quality.

* **Structured Scene Design**
  Organized objects efficiently using Blender’s collection system for better workflow.

## How to Run

1. Open the `.blend` file in Blender
2. Switch to **Layout / Animation workspace**
3. Press **Spacebar** to play the animation
4. Use **Camera View (Numpad 0)** to view animation properly
